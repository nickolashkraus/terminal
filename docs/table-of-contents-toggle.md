# Table of Contents Toggle

This theme provides functionality for toggling the table of contents similar to
toggling [NERDTree][NERDTree] in Vim/Neovim.

## What does this module do?

The `table-of-contents-toggle.js` JavaScript module adds toggling functionality
to the table of contents generated by Hugo. When a user presses CTRL + n, the
table of contents is hidden and the main content expands to fill its space.

## How does it work?

Hugo renders a table of contents when the `{{ .TableOfContents }}` template
code is used. The table of contents has the following form:

**HTML**

```html
<nav id="TableOfContents">
  <ul>
    <li><a href="#heading-1">Heading 1</a></li>
    <li><a href="#heading-2">Heading 2</a></li>
    <li><a href="#heading-3">Heading 3</a></li>
  </ul>
</nav>
```

**Rendered**  
Heading 1  
Heading 2  
Heading 3

By adding and removing specific TailwindCSS classes to the table of contents
element, empty spacer element, and main contents element, the table of contents
can be added or removed.

## Prerequisites

To use this module, you'll need:

1. Tailwind CSS
2. Add the following classes to the table of contents element, empty spacer
   element), and main contents element, the table of contents:

```html
<div class="hidden lg:col-span-2 lg:block togglable-toc">
  ...
</div>
<div class="hidden lg:col-span-1 lg:block togglable-empty"></div>
<div class="col-span-12 lg:col-span-9 togglable-main">
```

## Using the module

Import the module:

```javascript
import { initTableOfContentsToggle } from './modules/table-of-contents-toggle.js'
```

Then initialize it when your page loads:

```javascript
document.addEventListener('DOMContentLoaded', () => {
  initTableOfContentsToggle()
})
```

The module exports a single function, `initTableOfContentsToggle()`, which
sets up all necessary event listeners.
